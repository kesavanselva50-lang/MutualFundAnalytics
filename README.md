# Bluestock Mutual Fund Analytics

## Project Overview

Bluestock Mutual Fund Analytics is a financial analytics project developed to evaluate mutual fund performance, investor behavior, portfolio diversification, and risk metrics using Python, SQLite, and Power BI.

The project implements a complete analytics pipeline consisting of data extraction, transformation, risk analysis, performance evaluation, and dashboard visualization.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* SQLite
* Power BI
* Jupyter Notebook
* VS Code
